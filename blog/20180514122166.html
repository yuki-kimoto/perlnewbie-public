sin function - calculate the sine value

sin function calculates the sine value. The angle is specified by arc degree method using π (pi function). 

use Math::Trig 'pi';

sin(pi/2);

The pi function is defined in Math::Trig module.

sin function programming example

The following is a sin function programming example. 

use strict;
use warnings;
use Math::Trig 'pi';

# sin(90°) = 1;
print "sin(π/2) = " . sin(pi/2) . "\n";
