lc function - Convert uppercase string to lowercase string

lc function convert uppercase string to lowercase string. Each letter in the string is converted from uppercase to lowercase. 

my $str = lc "BBB";

lc function programing example

lc function to programing example. 

use strict;
use warnings;

# Convert uppercase string "BBB" to lowercase string "bbb"
my $str = lc "BBB";

print "$str\n";

lc function relative information

  • lcfirst function - Convert uppercase to lowercase
  • uc function - Convert lowercase to uppercase
  • ucfirst function - Convert lowercase first character of string to uppercase
